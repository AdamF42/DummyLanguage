ASSEGNAMENTO 3

"Realizzare un sistema di ANALISI SEMANTICA per l'estensione di SimpleStaticAnalysis con
dichiarazioni di funzioni e tipi di dato che si trova in allegato."

Questo progetto  utilizza il gestore dei pacchetti Maven

Prima dell'esecuzione del progetto modificare il file test.spl inserendo come input ciò che è stato richiesto nell'assegnamento

Per l'esecuzione del progetto bisogna:

1- scompattare lo zip in una cartella;
2- aprire Eclipse
3- andare su file > import > maven > "Existing Maven Projects";
4- selezionare la root del progetto (che contiene il file pom.xml);
5- andare su Run > run as > "maven install";
6- andare su Run > run as > java application e selezionare il main "esercizio1".
